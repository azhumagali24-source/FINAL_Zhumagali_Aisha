# Final Project: E-Learning Platform Database

## Domain Description
This database is designed for an Online Learning Platform. It manages instructors, course categories, educational courses, individual lessons, student registrations, course enrollments (Many-to-Many relationship), and financial payments.

## Database Details
- **Database Name:** `elearning_db`
- **Schema Name:** `elearning`

## How to Run the Script
1. Open DBeaver and connect to your PostgreSQL 17 server.
2. Create a clean database named `elearning_db`.
3. Open a **New SQL Editor** inside this database.
4. Copy the entire content of `02_final.sql` and paste it into the editor.
5. Press **Alt + X** (or click the orange "Execute SQL Script" button) to run the full file top-to-bottom.
6. To verify re-runnability, press **Alt + X** a second time. The script will clean up and recreate everything flawlessly.
